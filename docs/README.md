# Documentation — k8s-local-bench

This `docs/` folder contains user and developer documentation for the `k8s-local-bench` CLI.

Structure:

- `overview.md` — short project overview and quick start.
- `configuration.md` — in-depth configuration, env-vars, flags, and config-file behavior.
- `examples.md` — common workflows and reproducible command examples.
- `developer-notes.md` — notes for contributors and next steps.
- `commands/` — command-specific documentation:
  - `create.md` — `cluster create` deep dive
  - `destroy.md` — `cluster destroy` deep dive (status & implementation notes)

Start with `overview.md` then follow links to configuration and command pages.
